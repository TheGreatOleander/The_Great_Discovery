# The Great Discovery
### Roadmap

*This is not a feature list. It is a description of constraints narrowing — each phase ruling out more of what the engine cannot yet be, until what it must be settles into place.*

---

## How To Read This Roadmap

Features are added to products.
Constraints are discovered by engines.

This roadmap is not a promise of additions. It is a map of the settling process — each phase leaving the engine more structurally opinionated, more precise in its holes, closer to the configuration that the whole project implies but cannot yet fully occupy.

Nothing here will be forced.
If something won't settle, that is signal, not failure.

---

## Phase 4 — Recursive Discovery
*Status: 🔄 Active*

The mapmaker becomes the map.

**Implemented:**
- Question nodes participate in motif sampling and hole detection ✓
- Question-question edges (analogous_to, causes) create meta-topology ✓
- Meta-hole detection over question layer ✓
- Meta-questions asked — generate_meta_questions() wired, nodes injected at depth=1 ✓
- Recursion depth threading — depth=0 object questions, depth=1 meta-questions ✓
- AI augmentation layer (Phase 4.5) — Claude reads hole profiles, suggests settling concepts ✓

**Convergence system hardened (Phase 4 audit):**
- Five stability states: STABLE, DEADLOCKED, EXPANDING, DIVERGENT, OSCILLATORY
  - EXPANDING split from DEADLOCKED — flat compression + growing holes is hunger, not freeze
- DEADLOCKED response: perturbation node injection + temperature raise
- EXPANDING response: settling throttled to 1/epoch (let holes sharpen)
- DIVERGENT response: exploration temperature raised to 0.6
- OSCILLATORY response: temperature modulated in phase with detected period
- Mismatch governance: settling throttled at 0.25, suspended at 0.40
- Stress test assertions: 8 structural checks, exits non-zero on regression

**Open:**
- Depth=2 (meta-meta-questions) — infrastructure exists, rarely triggered
- Self-mapping completeness — the engine maps its question patterns but not yet
  the patterns of its own convergence states
- Named holes surfaced to external consumers — the question output is printed
  but not yet exposed as a structured API or feed

**This phase has no completion condition.**
It is the engine doing what it was always going to do, given enough time and a sealed enough system.

---

## What Will Never Be On This Roadmap

- A feature that forces knowledge into place
- A search function that retrieves known facts
- A recommendation engine that surfaces popular content
- A classifier that labels nodes with predetermined categories
- Any mechanism that tells the engine what to discover

The engine finds what the structure demands.
Or it finds nothing.
Both are valid outcomes.
Both are information.
