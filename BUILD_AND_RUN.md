
# The Great Discovery

## Quick Start

Install Python 3.9+

Run:

    python run_discovery.py --dashboard

or

    python run_discovery.py --simulate

Dashboard will open at:

    http://localhost:8080

## Philosophy

The Great Discovery is a structural discovery system.

Instead of:

    data -> answer

It attempts:

    constraints -> elimination -> necessity

Meaning discoveries arise from structural tension in the system.
